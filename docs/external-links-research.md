# ダム外部リンク調査結果

## 概要

ダム詳細ページに外部リンクセクションを追加するための調査結果をまとめたドキュメント。

| 項目                           | 数値         |
| ------------------------------ | ------------ |
| 全ダム数                       | 2,749基      |
| 川の防災情報カバー数           | 888基（32%） |
| 今回発見した公式サイト数       | 18サイト     |
| アプリ内ダムデータとのマッチ数 | 136基        |

## 方針

- 有志が作成したサイトではなく、自治体やダムの公式サイトのみ許可
- ダム詳細ページの対象ダムがサイトに実際に掲載されている場合のみリンクを表示
- 川の防災情報・雨雲レーダーは既存の位置に維持し、外部リンクセクションには含めない

## 注意事項

### ダム名の表記揺れ

アプリ内データと外部サイトでダム名の表記が異なる場合がある。

| アプリ内表記 | 外部サイト表記 | 差異     |
| ------------ | -------------- | -------- |
| 五ヶ山       | 五ケ山         | ヶ vs ケ |
| 耶馬溪       | 耶馬渓         | 溪 vs 渓 |

## 対象外サイト

### 東京都水道局

- **URL**: https://www.waterworks.metro.tokyo.lg.jp/suigen/suigen
- **除外理由**: 水系合計の貯水量のみ表示で、個別ダムの貯水率は表示されない
- **判断**: 外部リンクの対象としては優先度低（検討中）

### 北九州市上下水道局

- **URL**: https://www.city.kitakyushu.lg.jp/suidou/s00900011.html
- **除外理由**: 掲載施設が「貯水池」であり、アプリ内のダムデータと名称が一致しない
- **判断**: 外部リンクの対象外
