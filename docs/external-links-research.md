# ダム外部リンク調査結果

## 概要

ダム詳細ページに外部リンクセクションを追加するための調査結果をまとめたドキュメント。

| 項目                           | 数値         |
| ------------------------------ | ------------ |
| 全ダム数                       | 2,749基      |
| 川の防災情報カバー数           | 888基（32%） |
| 発見した公式サイト数           | 59サイト     |
| アプリ内ダムデータとのマッチ数 | 535基        |

## 方針

- 有志が作成したサイトではなく、自治体やダムの公式サイトのみ許可
- ダム詳細ページの対象ダムがサイトに実際に掲載されている場合のみリンクを表示
- 川の防災情報・雨雲レーダーは既存の位置に維持し、外部リンクセクションには含めない
