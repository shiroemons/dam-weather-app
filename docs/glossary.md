# 用語解説

## ダム名のサフィックス

アプリ内のダムデータには以下のサフィックスが付いているものがある。

### （元）・（再）

ダムの**再開発事業**に関連する区分。再開発とは、既存ダムの堤体嵩上げ・放流設備の増設・貯水池の掘削などにより、ダムの機能を向上させる事業のこと。

| サフィックス | 意味           | 説明                   |
| ------------ | -------------- | ---------------------- |
| （再）       | 再開発後のダム | 現在運用中の施設       |
| （元）       | 再開発前のダム | 再開発前の旧施設データ |

**例: 鶴田ダム**

- `鶴田（元）` — 竣工1965年。再開発前の旧ダムデータ
- `鶴田（再）` — 竣工2015年。再開発後の現行ダム

同じ場所にある同じダムだが、国交省のデータベース上では事業として区別されている。

### 付帯施設

メインダムに付随する施設。

| 名称     | 説明                                                                 |
| -------- | -------------------------------------------------------------------- |
| 副ダム   | 本ダムの下流に設置される小規模なダム。減勢（水の勢いを弱める）が目的 |
| 脇ダム   | ダム湖の周囲の低い尾根部分を塞ぐために設けられるダム                 |
| 鞍部ダム | 脇ダムと同様、ダム湖周囲の鞍部（尾根の低い部分）を塞ぐダム           |
| 調整池   | ダムから供給された水を一時的に貯留し、配水量を調整する施設           |

## 管理区分

ダムの管理者によって、貯水率情報の公開先が異なる。

| 管理区分       | 管理者                 | 主な情報公開先                   |
| -------------- | ---------------------- | -------------------------------- |
| 国管理（直轄） | 国土交通省の地方整備局 | 川の防災情報、各地方整備局サイト |
| 水資源機構管理 | 独立行政法人水資源機構 | 川の防災情報、水資源機構サイト   |
| 都道府県管理   | 各都道府県             | 都道府県の河川防災情報システム   |
| 市町村管理     | 各市町村               | 市町村の水道局サイト等           |
| 企業局管理     | 都道府県企業局         | 企業局サイト                     |

---

## 川の防災情報

国土交通省が運営するリアルタイム河川情報サービス（https://www.river.go.jp/）。

本アプリでは既にこのサービスからダムの貯水率データを取得しており（`riverUrl` フィールド）、ダム詳細ページのヘッダーにリンクを表示している。

---

## 貯水率

貯水率は、ダムの利水容量に対する現在の貯水量の割合を示す指標。

```
貯水率（%） = 現在の貯水量 ÷ 利水容量 × 100
```

- **利水容量**: 洪水調節容量を除いた、実際に水の供給に使える容量
- **総貯水容量**: ダムが貯められる水の総量（洪水調節容量を含む）

---

## 水系

同じ河川の本流と支流をまとめた河川群のこと。例えば「利根川水系」には、利根川本流のほか、渡良瀬川、鬼怒川、江戸川などの支流が含まれる。

ダムの管理や水の配分は水系単位で行われることが多い。
